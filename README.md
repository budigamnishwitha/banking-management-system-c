 💰 Banking Management System in C

📌 Project Overview

This is a console-based Banking Management System developed using the C programming language. It allows users to perform basic banking operations with persistent data storage using file handling.

🚀 Features

* Create new bank account
* Display all accounts
* Deposit money
* Withdraw money
* Search account details
* Prevent duplicate accounts
* Real-time balance updates
* Input validation and error handling

 🛠 Technologies Used

* C Programming Language
* File Handling (fopen, fread, fwrite)
* Structures

 ▶️ How to Run

1. Compile the program:
gcc main.c -o bank
2. Run the program:
./bank
